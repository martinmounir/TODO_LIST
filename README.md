# 📝 To-Do List Web App

A modern, responsive, and accessible To-Do List application built using **HTML, SASS/CSS, and Vanilla JavaScript**.

This project goes beyond a basic to-do list by focusing on **real-world practices** such as accessibility, clean architecture, and persistent data handling.

---

## 📌 Features

- ✅ Add, edit, and delete tasks
- 📱 Fully responsive (Mobile-First design)
- ♿ Accessibility (A11y) support
  - Keyboard navigation
  - Screen reader friendly
- 💾 Persistent data using **Web Storage API (Local Storage)**
- 🎯 Clean and intuitive UI/UX
- ⚡ Fast and lightweight (Vanilla JS only)

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **SASS / CSS3** – Styling & responsiveness
- **JavaScript (ES6+)** – Logic & interactivity
- **Local Storage API** – Data persistence

## ⚙️ How It Works

1. User adds a task
2. Task is stored in **Local Storage**
3. On page reload → tasks are reloaded automatically
4. User can update/delete tasks anytime
